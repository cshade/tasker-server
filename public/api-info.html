Tasker API Usage

Under /api/task

  • /all to retrieve all tasks
  • /random to retrieve a random task
  • /add to add a new task
  • /delete/<_id> -- e.g. /delete/21 to update task with model _id of 21
  • /update/<_id> -- e.g. /update/9 to update task with model _id of 9
  • Get /task/all

Under /api/reminder

  • /all to retrieve all reminders
  • /random to retrieve a random reminder
  • /add to add a new reminder
  • /delete/<_id> -- e.g. /delete/21 to update reminder with model _id of 21
  • /update/<_id> -- e.g. /update/9 to update reminder with model _id of 9
  • Get /reminder/all
